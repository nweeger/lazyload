What is this about?

gLazyLoad is a GwtQuery plugin to lazily load pictures on a webpage.

Instead of pictures being loaded with the page, they are only loaded when the user actually sees them.

This makes the page load faster for the user, and saves bandwidth on the server side.

This plugin is the GwtQuery equivalent of the jQuery LazyLoad plugin.

How to use it

Get the latest release, and add it to your project (the actual method depends on your IDE), which must already be using GwtQuery.

In your HTML code, replace the src attribute of your images with the data-original tag, add a lazy class.

Using a CSS rule or an inline style, make sure the pictures are correctly sized (using width: and height:), else the plugin may fail to work.

Import the package import static gwtquery.plugins.lazyload.client.LazyLoad.LazyLoad; in your Java code, then use $("img.lazy").as(LazyLoad).lazyload() to have pictures load lazily.

Example

In your HTML, put:

<img alt="picture" data-original="big-picture.png" style="width: 800px; height: 600px;" />

and in your Java code add, for instance in your module EntryPoint:

$("img").as(LazyLoad).lazyload();

When the page initially loads, the picture will be a blank rectangle. When the picture scrolls into view, the picture will be automatically loaded.

Detailed explanation:

  • the picture has a dimension of 800 pixels by 600, so we use a style="..." statement so the browser reserves space for the picture
  • $("img") selects all the pictures
  • .as(LazyLoad) converts the GwtQuery items to a LazyLoad plugin
  • .lazyload() applies lazy loading on the items with default parameters

Customizing

You may of course use any CSS class (rather than lazy) to refer to your pictures, or any other method, as long as you can get a GwtQuery instance on them.

You can customize the behaviour of the plugin by creating an instance of the Options class, calling the various setters, and giving it as a parameter to the .lazyload() function call.

In particular, you can specify:

  • the effect to apply to the picture when it is shown
  • the placeholder to use before the picture is loaded
  • the container the pictures are in, defaults to the whole client window

Available samples for gLazyLoad:

  • vertical scrolling
  • horizontal scrolling
  • fade in effect
  • vertical container
  • horizontal container
